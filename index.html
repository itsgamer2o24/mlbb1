📋 Setup Guide
  1. Create a GitHub repo (e.g. my-tournament) and enable GitHub Pages (Settings → Pages → Deploy from branch: main, /root)
  2. Upload index.html to the root of your repo
  3. Go to github.com/settings/tokens → Generate new token (classic) → check repo scope → copy the token
  4. Enter your details below and click Save Config
  5. All tournament data saves to data/db.json in your repo — any visitor can see it!
⚠️ Token stored only in your browser (localStorage). Never shared. Required for saving data.
